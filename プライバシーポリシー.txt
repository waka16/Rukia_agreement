Team waka16.com プライバシーポリシー

はじめに
	Team waka16.com（以下「当団体」といいます）は、「Rukia」（以下、「本サービス」と言います）をご利用になる会員様（以下「ユーザー」と言います。）のプライバシーを尊重し、ユーザーの個人情報の管理に細心の注意を払うものとします。

1 適用範囲
	プライバシーの適用範囲はRukiaに投稿していただいている情報や登録されている情報に適用されます。Rukiaがサービスを提供するために収集された個人情報はプライバシーの考え方に従って管理されます。

2 情報の収集と利用
	本サイトはユーザのIPアドレス、クッキー情報、ユーザが閲覧したページ、ユーザの利用環境等の情報をユーザのブラウザから自動的に受け取り、サーバーに記録します。
	当団体は、ユーザーが本サービスを利用することで取得した個人情報を次の目的のために利用させて頂きます。
		( 1 ) サービスの内容をより良いものにするため。
		( 2 ) ユーザーに新しいサービスや情報を的確にお届けするため。
		( 3 ) 必要に応じてユーザーに連絡をするため。
		( 4 ) ユーザーにより良い環境を提供するため。

3 情報の開示
	当団体はユーザーにより入手した個人情報を販売・貸出をしません。
	ただし、以下の場合、当団体は個人情報を開示することがあります。 当団体は、ユーザーが本サービスを利用することで取得した個人情報について、以下に掲げる場合を除き、上記利用目的の達成に必要な範囲内において使用するものとします。

		( 1 ) 裁判所や警察等の公的機関から、法令に基づく正式な照会要請を受けた場合
		( 2 ) 人の生命、身体及び財産等に対し差し迫った危険があり、緊急の必要性があると認められる場合
		( 3 ) ユーザーが希望するサービスの提供のため、情報の開示や共有が必要と認められる場合
		( 4 ) ユーザーにサービスを提供する目的でRukiaから委託を受けて業務を行う会社や団体が情報を必要とする場合
		( 5 ) 本サービス上でのユーザーの行為が、利用規約に反し、当団体の権利・財産やサービス等を保護するため必要と認められる場合

4 免責
	次に掲げる場合には、第三者による個人情報の取得に関し、当社は一切の責任を負いません。
		( 1 ) ユーザーご本人が当社サービスの機能または別の手段を用いて第三者に個人情報を明らかにした場合
		( 2 ) ユーザーが当社サービス上に入力した情報等により、個人が識別できてしまった場合

5 プライバシーポリシーの改正
	今後当団体はプライバシーポリシーの全部または一部を改定することがあります。
